单调性

首先发现答案具有单调性，可以二分求解。

十分容易证明，如果现在的速度是 xxx，那么对于答案大于 xxx 的情况他们完全可以沿用速度为 xxx 的方案跑完。

不必要的情况

接下来跟着官方题解手玩输入样例，发现了一下一些情况：

  1. 对于一样的场面，似乎只改变 TTT 也会影响答案的方案。
  2. 有些人可能会停下来。
  3. 有很多人的烟花在被同时点燃。

但是其实这些情况都是没有必要的，似乎有多人做题的时候被误导了（但是学校模拟赛没有给样例解释导致题目没有看懂）。

情况 111

显然会有一种通用的方式，如果这个方式可以满足较小的 TTT 那么也一定可以满足较大的 TTT。

具体的，让本会熄灭的人拿着烟花在站着就行了。

情况 222

显然，停止是不必要的。

具体的，如果我们要停止 ttt 秒那么我们可以向相反方向前进 t2\dfrac{t}{2}2t​ 就行了。

因为我们考虑的是相对位置，所以整体一起向一个方向移动移动可是一种思路。

情况 333

只考虑 222 个的情况，其他的情况容易拓展。

考虑分情况讨论：

  • 如果 222 个人相遇后向同一方向运动，那么显然我们可以让其中 111 个人点火之后一起跑，等第 111 个人熄灭之后再传递给第 222 个人。这样有火的时间就从 TTT 变成了 2T2T2T。
  • 如果 222 个人相遇之后向相反方向移动，那么我们考虑延迟第 222 个人的点时间。为了不影响相对位置，我们可以让第 111 个人先点火，然后让与第 222 个人有关的人一起与第 111 个人运动，保持相对位置不要变。这样当第 111 个人的火将要熄灭的时候第 222 个人再点燃。

规律

  1. 因为情况 333，场上最多只有 111 个人的火把被点燃。
  2. 没点燃的人不会主动远离已经点燃的人，也就是会主动靠近被点燃的人。
  3. 拿着烟花的人应该主动靠近下一个需要被点燃的人，这十分显然。

也就是说如果确定了点燃的顺序，那么点燃的最有方案就是确定的。

所以容易想到 O(n×n!×log⁡V)O(n\times n!\times \log V)O(n×n!×logV) 的做法，可惜题目没有给分。

可行性优化

如果有 333 个人 x,y,zx,y,zx,y,z 顺序排列，那么按照 x→z→yx\to z\to yx→z→y 的顺序点火肯定没有按照 x→y→zx\to y\to zx→y→z 或者 z→y→xz\to y\to xz→y→x 的顺序点火优秀。

因此对于任意时刻，被点燃的火把一定是一个包含 kkk 的连续区间。

那么我们就只需要讨论两侧的情况就行了，时间复杂度为 O(n×2n×log⁡V)O(n\times 2^n\times \log V)O(n×2n×logV)，这样可以获得 303030 分。

最优性优化

因为我们发现点燃的一定是一个包含 kkk 的连续的区间，那么容易想到将最初的 [k,k][k,k][k,k] 逐步向外拓展到 [1,n][1,n][1,n]。
