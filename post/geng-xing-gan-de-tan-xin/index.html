Playing Around the Table

题目大意

有 nnn 种牌，一种 nnn 张，一共 nnn 个玩家，一人 nnn 个。每个人一次将一张牌对给下家，求构造方案使可以在 n⋅(n−1)n\cdot (n-1)n⋅(n−1) 次操作之内使第 iii 个人拥有 nnn 张 iii。

数据范围满足，1≤n≤1001\le n\le 1001≤n≤100。

思路

因为直接构造出题目要求的情况会出现如果一个人提前完成了牌的收集，那么在接下来的操作中他的牌会被打乱，并不足够优秀。

考虑先构造出每一个人都拥有 111 到 nnn 号牌的方案，在使用 n⋅(n−1)2\dfrac{n\cdot(n-1)}{2}2n⋅(n−1)​ 的交换次数交换为题目要求的方案。

现在问题转化为了怎么构造出每一个人都拥有 111 到 nnn 号牌的方案。

如果没有构造出上述的情况，那么一定有一个人拥有重复的牌，将这个牌传给下家。如果这个人手中的牌都没有重复，那么可以它可以将上家传给他的牌传给下家，这样就可以保证手中现在的情况不被破坏。

只考虑值为 111 的牌需要被往前送多少次。把牌看作 111 ，人看作 −1−1−1 ，那么必然存在一个循环移位使得前缀和非负。于是不难发现，在这个循环移位下，每张牌只会往前走，不会从第 nnn 个人回到第 111 个人。

所以这个操作是可以在 n⋅(n−1)2\dfrac{n\cdot(n-1)}{2}2n⋅(n−1)​ 次操作之内完成的。

Winter 2021 Day4I Color

题目大意

给定一个 nnn 个点的完全图，每条边都被染了 mmm 种颜色中的一种。

你需要判断能否再加入 m−n+1m − n + 1m−n+1 个点，补成一个 m+1m + 1m+1 个点的完全图，并且每条边仍然被染了 mmm 种颜色中的一种，使得每个点都满足相邻的边的颜色恰好取遍这 mmm 种。

思路

先把原图不合法和 mmm 是偶数的情况判断掉。

在最终的图中会有一条边会有 333 种情况：

  • 对于两个端点都已经确定的情况，我们把这样的边加入这个颜色的 2-set。
  • 对于有一个端点确定的情况，我们把这样的边加入这个颜色的 1-set。
  • 对于有没有端点确定的情况，我们把这样的边加入这个颜色的 0-set。

如果一个颜色的 1-set 和 2-set 的大小加起来比 m+12\dfrac{m+1}{2}2m+1​ 还要大，那么显然是无解的。

否则，我们可以考虑证明其一定是有界的，具体的只需要证明可以新加入一个点然后是合法的就行了。

那么容易想到用二分图跑最大流来进行这个拓展：

左边放 mmm 个点表示 mmm 种颜色，右边放 n+1n+1n+1 个点，前 nnn 个点表示和新加入的点的连边颜色，而最后面的点是还没有考虑的点与新加的点匹配。

如果第 xxx 个点周围没有颜色 iii，那么就把颜色 iii 与 xxx 点连接为流量为 111 的边。

如果 x,ix,ix,i 匹配那么就代表新加入的点要与 xxx 连接一个颜色为 iii 的边。

另外每个颜色还要和右边的第 n+1n+1n+1 个点连接一条流量为 111 的边，代表新加的这个点的这个颜色以后再匹配。

特殊的，第 n+1n+1n+1 个点向汇点的流量是 m−nm-nm−n，根据其定义容易理解。

在跑了网络流之后去更新 333 个 set 的数据就行了。

显然如果我们跑了满流，那么就代表我们成功的拓展了一次，但是似乎很难证明这个东西一定是满流。

我们引入了一个牛逼定理（pb 都不会证的那种），就是每个点的都都一样的二分图一定可以跑满流。

那么因为每一个节点的流量最大都只有 111，那么如果我们可以在中间随便多连一些边或者把一些点拆爆然后乱搞去凑上面的定理。

pb 使用的方法是让所有的颜色都给右边 n+1n+1n+1 连 222 倍 0-set 的点，这样出去右边 n+1n+1n+1 所有的点都是 m−n+1m-n+1m−n+1 个出边。

发现左边的节点是 m−n+1m-n+1m−n+1 有点不好证，考虑对于一种颜色，把其边的对应的端点用表格的方式表现出来，不知道的留空。

那么 1−set1-set1−set 有 111 个空子，2−set2-set2−set 没有空子，0−set0-set0−set 有两个空子，所以空子的数量就是 2×S0+S12\times S_0+S12×S0​+S1。

用另一种角度考虑，其实这个空子的数量也是 m−n+1m-n+1m−n+1 因为格子里面一共加入了 nnn 个点而一共有 m+1m+1m+1 个点，于是得证。

这个时候右边 n+1n+1n+1 的点连了 (m−n)×(m−n+1)(m-n)\times (m-n+1)(m−n)×(m−n+1) 条边，那么我们直接把这个点拆爆成 m−nm-nm−n 个，这样我们就凑出了牛逼定理，也就证明了做法的正确性。

Empty up a Bottle

容易发现，进行一次操作相当于：

A←2A mod (A+B)B←2B mod (A+B)\begin{aligned} A\gets 2A\bmod (A+B)\\ B\gets 2B\bmod (A+B) \end{aligned}A←2Amod(A+B)B←2Bmod(A+B)​

如果 A+BA+BA+B 为奇数，就有：

2φ(A+B)≡1(modA+B)2^{\varphi(A+B)}\equiv 1\pmod{A+B} 2φ(A+B)≡1(modA+B)

于是我们就有了除法，所以就很牛逼。

我们不妨假设 A,BA,BA,B 为偶数，CCC 为奇数，对于其他的情况我们都可以将其转化为这种情况。

假设都是奇数，那么随便找两个处理一下就是我们需要的情况。

假设都是偶数，那么我们假设他们全部除以 222，不进行操作，因为全部乘以倍数并不影响我们后面的构造。

在得到希望的情况之后，不妨假设 B>AB>AB>A，我们有构造：

{A,B,C}→{2A,B−A,C}→{A,B−A,C+A}\{A,B,C\}\to \{2A,B-A,C\}\to\{A,B-A,C+A\} {A,B,C}→{2A,B−A,C}→{A,B−A,C+A}

如果没有次数限制，那么我们反复进行上述操作，用类似于辗转相减法一定可以最后得到 000。

但是如果 BBB 远大于 AAA，500500500 次肯定不够用，于是我们考虑在让 BBB 减去 2k2^k2k，加速处理。

那么我们有更新的构造：

{A,B,C}{2kA mod (A+C),B,2kC mod (A+C)}{2k+1A mod (A+C),B−2kA,2kC mod (A+C)}{2kA,B−2kA,C−2k+1A mod (A+C)}{A,B−2kA,C2k−2 mod (A+C)}\begin{aligned} &\{A,B,C\}\\ &\{2^kA\bmod (A+C),B,2^{k}C\bmod (A+C)\}\\ &\{2^{k+1}A\bmod (A+C),B-2^{k}A,2^kC\bmod (A+C)\}\\ &\{2^{k}A,B-2^kA,C-2^{k+1}A\bmod (A+C)\}\\ &\{A,B-2^kA,\dfrac{C}{2^k}-2\bmod (A+C)\} \end{aligned}​{A,B,C}{2kAmod(A+C),B,2kCmod(A+C)}{2k+1Amod(A+C),B−2kA,2kCmod(A+C)}{2kA,B−2kA,C−2k+1Amod(A+C)}{A,B−2kA,2kC​−2mod(A+C)}​

显然 CCC 的值会一直保持为奇数，因为 333 个数的和是一定的，A,BA,BA,B 始终为偶数。
