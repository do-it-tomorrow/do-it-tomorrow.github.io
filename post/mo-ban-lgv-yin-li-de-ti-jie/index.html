其中 SSS 表示表示从 AAA 到 BBB 所有的不相交的路径 PPP。

首先考虑行列式的定义，即对于一个 n×nn\times nn×n 的方阵 AAA，其行列式为：

其中 σ\sigmaσ 是一个 111 到 nnn 的排列。

具体到上面的矩阵，显然其行列式为：

Det⁡(M)=∑∀σ(−1)t(σ)∏i=1ne(Ai,Bσ(i))\operatorname{Det}(M)=\sum\limits_{\forall \sigma}(-1)^{t(\sigma)}\prod\limits_{i=1}^n e(A_i,B_{\sigma(i)}) Det(M)=∀σ∑​(−1)t(σ)i=1∏n​e(Ai​,Bσ(i)​)

根据 e(x,y)e(x,y)e(x,y) 的定义，其显然可以转化为：

Det⁡(M)=∑∀σ(−1)t(σ)∏i=1n∑∀P:Ai→Bσ(i)ω(P)\operatorname{Det}(M)=\sum\limits_{\forall \sigma} (-1)^{t(\sigma)}\prod\limits_{i=1}^n \sum\limits_{\forall P:A_i\to B_{\sigma(i)}}\omega(P) Det(M)=∀σ∑​(−1)t(σ)i=1∏n​∀P:Ai​→Bσ(i)​∑​ω(P)

根据乘法原理，上式的 ∏i=1n∑∀P:Ai→Bσ(i)ω(P)\prod\limits_{i=1}^n \sum\limits_{\forall P:A_i\to B_{\sigma(i)}}\omega(P)i=1∏n​∀P:Ai​→Bσ(i)​∑​ω(P) 就相当求解的是在所有的 ∀i∈[1,n]∩Z∣Ai→Bσ(i)\forall i\in[1,n]\cap \mathbb {Z}\mid A_i\to B_{\sigma(i)}∀i∈[1,n]∩Z∣Ai​→Bσ(i)​ 的路径中，所有可能的路径的组 PPP 的 ∑i=1nω(Pi)\sum\limits_{i=1}^n \omega(P_i)i=1∑n​ω(Pi​) 的加和。

所以就可以继续化简得到：

Det⁡(M)=∑∀σ(−1)t(σ)∑P=σω(P)\operatorname{Det}(M)=\sum\limits_{\forall \sigma} (-1)^{t(\sigma)} \sum\limits_{P=\sigma} \omega(P) Det(M)=∀σ∑​(−1)t(σ)P=σ∑​ω(P)

还是根据乘法原理，有可以得到：

Det⁡(M)=∑∀P:A→B(−1)t(σ)∏i=1nω(Pi)\operatorname{Det}(M)=\sum\limits_{\forall P\color{black}:A\to B} (-1)^{t(\sigma)}\prod\limits_{i=1}^n \omega(P_i) Det(M)=∀P:A→B∑​(−1)t(σ)i=1∏n​ω(Pi​)

观察上式与定理的区别，发现现在仅需证明对于所有有交的路径 GGG 的贡献均为 000 就可以证明原定里了。

要证明原定理成立，首先需要证明对于一个排列 σ\sigmaσ 满足交换 i,j∈[1,n]∩Z∧i≠ji,j\in[1,n]\cap \mathbb{Z}\wedge i\ne ji,j∈[1,n]∩Z∧i​=j 一定会使 σ\sigmaσ 逆序对的奇偶性改变。

先证明交换相邻的元素序列的奇偶性会改变，这十分显然因为如果交换 Ai,Ai+1A_i,A_{i+1}Ai​,Ai+1​ 那么因为没有相同的元素所以 Ai,Ai+1A_i,A_{i+1}Ai​,Ai+1​ 之间必定会贡献一个逆序对的变化，而对于 [1,i)[1,i)[1,i) 和 (i+1,n](i+1,n](i+1,n] 的贡献都是一样的，所以 t(σ)t(\sigma)t(σ) 的奇偶性必然改变。考虑更一般的情况，假设只能交换相邻的元素，交换 AiA_iAi​ 与 AjA_jAj​ 的贡献就是交换 Ai,Aj−1A_i,A_j-1Ai​,Aj​−1 的贡献 +2+2+2（一开始把 AjA_jAj​ 换到 Aj−1A_{j-1}Aj−1​ 最后再换回来）。

我们可以一直将距离不断缩小直到 j=i+1j=i+1j=i+1，因为我们已经证明了交换 Ai,Ai+1A_i,A_{i+1}Ai​,Ai+1​ 会改变奇偶性所以要自然可以证明一个排列任意交换两个元素其逆序对的奇偶性偶会改变。

对于一个有交点的路径 Ai→Bi,Aj→BjA_i\to B_i,A_j\to B_jAi​→Bi​,Aj​→Bj​，假设其第一次相交于 uuu，也就是其路径形如 Ai→u→BiA_i\to u\to B_iAi​→u→Bi​ 和 Aj→u→BjA_j\to u\to B_jAj​→u→Bj​。这时候如果交换 σ(i)\sigma(i)σ(i) 和 σ(j)\sigma(j)σ(j) 那么就会发现另外一条相交的路径 Ai→u→BjA_i\to u\to B_jAi​→u→Bj​ 和 Aj→u→BiA_j\to u\to B_iAj​→u→Bi​。

本题的求解

因为这个题目保证 1≤a1≤a2≤⋯≤am≤n1\leq a_1\leq a_2\leq \dots\leq a_m\leq n1≤a1​≤a2​≤⋯≤am​≤n，1≤b1≤b2≤⋯≤bm≤n1\leq b_1\leq b_2\leq \dots\leq b_m\leq n1≤b1​≤b2​≤⋯≤bm​≤n，所以如果起点与终点并不是一一对应的，那么就一定会出现交点，也就是说有贡献的 σ\sigmaσ 都满足 σ(i)=i\sigma(i)=iσ(i)=i。

对于 e(Ai,Bj)e(A_i,B_j)e(Ai​,Bj​) 使用组合数求解，直接输出 Det⁡(M)\operatorname{Det}(M)Det(M) 就可以了，时间复杂度 O(n3)O(n^3)O(n3)。

#include<iostream>
#define int long long 
using namespace std;
const int N=5e6+5,M=505,mod=998244353;
int ksm(int a,int b){
	int ans=1;
	while(b){
		if(b&1){
			ans=ans*a%mod;
		}
		b>>=1;
		a=a*a%mod;
	}
	return ans;
}
int n,m,a[M],b[M],inv[N],jc[N],f[M][M];
int C(int n,int m){
	if(n<m){
		return 0;
	}
	return jc[n]*inv[m]%mod*inv[n-m]%mod;
}
void solve(){
	cin>>n>>m;
	for(int i=1;i<=m;i++){
		cin>>a[i]>>b[i];
	}
	for(int i=1;i<=m;i++){
		for(int j=1;j<=m;j++){
			f[i][j]=C(b[j]-a[i]+n-1,n-1);
		}
	}
	int flag=1;
	for(int i=1;i<=m;i++){
		for(int j=i+1;j<=m;j++){
			while(f[i][i]){
				int s=f[j][i]/f[i][i];
				for(int k=i;k<=m;k++){
					f[j][k]=(f[j][k]-s*f[i][k]+mod)%mod;
				}
				flag++;
				swap(f[i],f[j]);
			}
			flag++;
			swap(f[i],f[j]);
		}
	}
	int ans=1;
	for(int i=1;i<=m;i++){
		ans=(ans*f[i][i])%mod;
	}
	cout<<(ans*(flag%2?1:-1)+mod)%mod<<'\n';
}
signed main(){
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	jc[0]=1;
	for(int i=1;i<N;i++){
		jc[i]=jc[i-1]*i%mod;
	}
	inv[N-1]=ksm(jc[N-1],mod-2);
	for(int i=N-1;i>=1;i--){
		inv[i-1]=inv[i]*i%mod;
	}
	int T;cin>>T;
	while(T--) solve();
	return 0;
}
