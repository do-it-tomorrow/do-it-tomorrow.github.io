AHOI 2017/HNOI 2017 大佬

· 2024-11-23 · # 动态规划 # 题解

发现这 555 个操作很神奇，最重要的是攻击力与操作的时机没有关系，而且攻击力和血量之间并没有联系。

具体的，发现操作 222 操作回血，其他操作都是攻击，那么考虑先只回血。

那么现在就进行一个简单 DP 求出在这 nnn 天中需要用多少天时间来回血才不会没大佬干掉，所以问题就转化为了剩余 ttt 天能否刚好的大佬干掉。

考虑暴力 BFS 求出用 ddd 天是否可以打掉 fff 中伤害，那么按照 fff 从小到大排序之后可以得到以下不等式：

fi+fj≤C∧fi+fj+t−(xi+xj)≥Cf_{i}+f_{j}\le C\wedge f_i+f_j+t-(x_i+x_j)\ge C fi​+fj​≤C∧fi​+fj​+t−(xi​+xj​)≥C

也就是使用技能攻击不会把大佬打成负数而且平 A 刚好可以把大佬刮死。

发现固定 iii 之后可以双指针维护 jjj。

对于 (d,f)(d,f)(d,f) 这个二元组需要使用哈希判重。
