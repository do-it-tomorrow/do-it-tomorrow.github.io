CF1864E Guess Game

· 2024-10-24 · # Codeforces # 题解

模拟赛考的，先手是知更鸟，后手是旅行者。

注意按位与有什么特点，假设现在是知更鸟：

  • xi=0,zi=1x_i=0,z_i=1xi​=0,zi​=1 那么有 yi=1y_i=1yi​=1 也就是 x<yx\lt yx<y。
  • xi=1,zi=1x_i=1,z_i=1xi​=1,zi​=1 那么 yiy_iyi​ 是不确定的，考虑继续让旅行者去考虑。
    • 因为知更鸟并不知道 yiy_iyi​ 是什么，那么 xix_ixi​ 就肯定是 111，所以旅行者就可以推断出知更鸟是什么。
    • 如果 yi=0y_i=0yi​=0 那么可以确定 x>yx\gt yx>y。
    • 否则 i←i−1i\gets i-1i←i−1，继续进行第一个操作。

发现就是统计第一个 010101 或者 101010 的前面有多少个 111111，可以 n2n^2n2 枚举然后 log⁡V\log VlogV 判断，可以拿 606060 分。
