在使用上述公式的情况下，不管可以计算凸多边形的面积，在给整体添加了绝对值之后还可以计算凹多边形的面积。另外可以发现，如果所有的点都是整点，那么 2S2S2S 一定是一个整数。

signed main(){
	ios::sync_with_stdio(false);

	cin>>n;
	for(int i=1;i<=n;i++){
		a[i].input();
	}
	db ans=0;
	for(int i=1;i<=n;i++){
		ans+=a[i]%a[i%n+1];
	}
	if(sgn(ans)==-1){
		ans=-ans;

	print(ans/2.0);
	cout<<'\n';
	return 0;
}

判断多边形的凹凸

首先我们不可以直接计算两个向量的夹角，因为我们并不知道我们计算的夹角是位于多边形之内还是多边形外侧。所以我们可以将连续的 333 个点一起考虑，得到这样的两个向量：

在上图中得到的两个向量是 AB→\overrightarrow{AB}AB 和 AC→\overrightarrow{AC}AC，接着计算 AB→×AC→\overrightarrow{AB}\times \overrightarrow{AC}AB×AC 判断 <AB→,AC→><\overrightarrow{AB},\overrightarrow{AC}><AB,AC> 是否大于 180°180\degree180°。具体的说，如果 AB→×AC→≥0\overrightarrow{AB}\times \overrightarrow{AC}\ge 0AB×AC≥0 那么就符合要求了。

对于下面的这个凹多边形会在访问到 DDD 点是被判断掉。

因为 <DE→,DA→><\overrightarrow{DE},\overrightarrow{DA}><DE,DA> 是 DE→\overrightarrow{DE}DE 逆时针旋转到 DA→\overrightarrow{DA}DA 大于 180°180\degree180°，所以 DE→×DA→\overrightarrow{DE}\times \overrightarrow{DA}DE×DA 是负数也就发现是凹多边形了。
