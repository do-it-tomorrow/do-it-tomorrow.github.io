[AGC009E] Eternal Average 的题解

· 2024-09-22 · # Atcoder # 题解

发现题目将 kkk 个数取平均数后合并成一个数的操作可以看作一棵 kkk 叉树，每一个节点的儿子数都是 000 或 kkk。

其中叶子节点是 000 或 111 表示给定的 n+mn+mn+m 个数，而非叶子节点的值就是其儿子的平均值，代表在操作后新产生的点。

假设所有的叶子节点从左到右排列的值依次是 a1,a2,⋯ ,an+ma_1,a_2,\cdots ,a_{n+m}a1​,a2​,⋯,an+m​，那么对于第 iii 个叶子节点其贡献可以写作 aikdep(i)\dfrac{a_i}{k^{dep(i)}}kdep(i)ai​​，其中 dep(i)dep(i)dep(i) 表示第 iii 个叶子节点的深度（dep(rt)=0dep(rt)=0dep(rt)=0）。所以在确定了 aaa 数组之后，剩余平均值就是 ∑i=1naikdep(i)\sum\limits_{i=1}^n \dfrac{a_i}{k^{dep(i)}}i=1∑n​kdep(i)ai​​。

设 ddd 为有 n+mn+mn+m 个点的 kkk 叉树的深度的最大值，容易得到 d=n+m−1k−1d=\dfrac{n+m-1}{k-1}d=k−1n+m−1​。

因为对于 ∀i∈[1,n+m]∩N\forall i\in[1,n+m]\cap\mathbb{N}∀i∈[1,n+m]∩N 满足 kdep(i)∣kdk^{dep(i)}\mid k^{d}kdep(i)∣kd，所以答案答案一个可以写成 Skd\dfrac{S}{k^{d}}kdS​ 的形式，其中 SSS 是一个整数。

发现直接统计答案会有重复，因为如果第 xxx 层有 kkk 个 111，那么其贡献等于在 x−1x-1x−1 层有 111 个 111，考虑将这 kkk 个 111 合并上去。注意，这 kkk 个 111 的父亲不一定是相同的。

考虑对于所有一层出现的 111 超过 kkk 次的情况全部处理掉，假设处理了 lll 次，那么 m′←m−l×(k−1),d′←d−lm'\gets m-l\times(k-1),d'\gets d-lm′←m−l×(k−1),d′←d−l。

注意 d′d'd′ 表示的 n+m′n+m'n+m′ 个点可以组的 kkk 叉树的最大深度，虽然可以构造出进行一次操作之后树的深度不减小的情况，但是这一定不是深度最深的情况。

如果将 SSS 看作一个 ddd 进制数，上面的操作就相当于给 SSS 进行进位操作。

假设现在的答案可以写作 S′kd′\dfrac{S'}{k^{d'}}kd′S′​，另 vvv 将 S′S'S′ 写作 d′d'd′ 进制数之后各位的值的和，那么需要满足 v=m′v=m'v=m′，也就是满足 v≡m(modk−1)v \equiv m \pmod{k-1}v≡m(modk−1)。

考虑使用 DP 进行求解方案数，设 fi,jf_{i,j}fi,j​ 表示一共 iii 层一共选择了 jjj 个 111 的方案数，模拟背包直接进行转移即可，时间复杂度为 O(n2)O(n^2)O(n2)。
