欧拉函数

欧拉函数 φ(n)\varphi (n)φ(n) 定义为小于或等于 nnn 且与 nnn 互质的正整数的个数。

欧拉函数之所以需要学，就是因为它又一些奇奇怪怪的性质：

  1. 如果 ppp 是质数，那么 φ(p)=p−1\varphi(p)=p-1φ(p)=p−1。
  2. φ\varphiφ 是积性函数，即如果 gcd⁡(a,b)=1\gcd(a,b)=1gcd(a,b)=1 那么 φ(a⋅b)=φ(a)⋅φ(b)\varphi(a\cdot b)=\varphi(a)\cdot \varphi(b)φ(a⋅b)=φ(a)⋅φ(b)。
  3. 对于任意 nnn 满足 ∑d∣nφ(d)=n\sum_{d\mid n}\varphi(d)=n∑d∣n​φ(d)=n，a∣ba|ba∣b 的意思是 gcd⁡(a,b)=a\gcd(a,b)=agcd(a,b)=a。

不定方程

不定方程可以使用 exgcd⁡\operatorname{exgcd}exgcd 进行求解，先给代码。


int exgcd(int a,int b,int &x,int &y){
	if(b==0){

    int ans=exgcd(b,a%b,y,x);

exgcd⁡\operatorname{exgcd}exgcd 函数可以求解方程 ax+by=gcd⁡(a,b)ax+by=\gcd(a,b)ax+by=gcd(a,b)（a,ba,ba,b 为常数）的整数根 x,yx,yx,y。

但是如果需要求解 ax+by=cax+by=cax+by=c，那么我们可以先求解 ax+by=gcd⁡(a,b)ax+by=\gcd(a,b)ax+by=gcd(a,b)，接着将将 x,y,gcd⁡(a,b)x,y,\gcd(a,b)x,y,gcd(a,b) 全部乘以 cgcd⁡(a,b)\dfrac{c}{\gcd(a,b)}gcd(a,b)c​ 就可以了。

一般性同余方程

对于同余方程 ax≡b(modn)ax \equiv b \pmod nax≡b(modn)，我们需要找到一个整数 xxx，使得 axaxax 除以 nnn 的余数为 bbb。这个问题可以通过以下步骤解决：

  1. 计算 gcd⁡(a,n)\gcd(a,n)gcd(a,n)：首先，我们需要计算 aaa 和 nnn 的最大公约数 ddd。这可以通过欧几里得算法实现。

  2. 检查 bbb 是否是 ddd 的倍数：如果 bbb 不是 ddd 的倍数，那么这个同余方程无解。因为如果 axaxax 除以 nnn 的余数为 bbb，那么 bbb 必须是 aaa 的倍数。但是，如果 bbb 是 ddd 的倍数，那么我们可以继续下一步。

  3. 求解同余方程：然后，我们需要求解同余方程 ax′≡b′(modn′)ax' \equiv b' \pmod {n'}ax′≡b′(modn′)，其中 x′=x/dx' = x/dx′=x/d，b′=b/db' = b/db′=b/d，n′=n/dn' = n/dn′=n/d。

  4. 找到最小的正整数解：最后，我们需要找到最小的正整数解。这可以通过对x取模n来实现。

因为这可能不好实现，所以我封装了一个函数。


long long extended_gcd(long long a, long long b, long long &x, long long &y) {
    if (b == 0) {
        x = 1;
        y = 0;
        return a;
    }
    long long x1, y1;
    long long gcd = extended_gcd(b, a % b, x1, y1);
    x = y1;
    y = x1 - (a / b) * y1;
    return gcd;
}
long long solve_congruence(long long a, long long b, long long n) {
    long long x, y;
    long long gcd = extended_gcd(a, n, x, y);
    if (b % gcd != 0) {
        return -1;  // 无解
    }
    x *= b / gcd;
    n /= gcd;
    x = (x % n + n) % n;  // 转化为最小的正整数解
    return x;
}

逆元

众所周知同余有很多性质但是都不适用于除法，所以在取模之后就不能使用除法了。

假设你到计算 xa\dfrac{x}{a}ax​ 在对 modmodmod 取模意义下的结果，那么你就不可以直接进行除法操做，而是要计算 x⋅inv⁡(a)%modx\cdot \operatorname{inv}(a)\% modx⋅inv(a)%mod 的结果，其中 inv⁡\operatorname{inv}inv 表示逆元。

形式化的 a⋅inv(b)⁡≡ab(modm)oda\cdot \operatorname{inv(b)} \equiv \dfrac{a}{b} \pmod moda⋅inv(b)≡ba​(modm)od。

在 b,modb,modb,mod 互质时可以使用 「NOIP2012」同余方程 中的扩展欧几里得定理进行求解。

在 modmodmod 为质数是可以使用费马小定理求解，具体来说 inv(b)⁡=bmod−2 %mod\operatorname{inv(b)}=b^{mod-2}\ \%modinv(b)=bmod−2 %mod，可以使用快速幂进行求解。


int inv(int a){
    int ans=1,b=mod-2;

int CRT(int k, int* a, int* r) {
  int n = 1, ans = 0;
  for (int i = 1; i <= k; i++) n = n * r[i];
  for (int i = 1; i <= k; i++) {
    int m = n / r[i], b, y;
    exgcd(m, r[i], b, y);  // b * m mod r[i] = 1
    ans = (ans + a[i] * m * b % n) % n;

}
